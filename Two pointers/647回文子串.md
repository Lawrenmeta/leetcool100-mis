使用回文串一般的dp解法，注意的是从小到大dp时的启动条件边界问题，这题里的代码是<3。

## 题目

给你一个字符串 `s` ，请你统计并返回这个字符串中 **回文子串** 的数目。、

**回文字符串** 是正着读和倒过来读一样的字符串。

**子字符串** 是字符串中的由连续字符组成的一个序列。

**示例 1：**

```
输入：s = "abc"
输出：3
解释：三个回文子串: "a", "b", "c"
```

**示例 2：**

```
输入：s = "aaa"
输出：6
解释：6个回文子串: "a", "a", "a", "aa", "aa", "aaa"
```

**提示：**

- `1 <= s.length <= 1000`
- `s` 由小写英文字母组成

## 代码

```cpp
class Solution {
public:
    int countSubstrings(string s) {
        int len=s.size();
        int cont=0;
        vector<vector<bool>> dp(len,vector<bool>(len,false));
        for(int i=0;i<len;i++){
            dp[i][i]=true;
            cont++;
        }
        for(int l=2;l<=len;l++){
            for(int i=0;i<len;i++){
                int j=i+l-1;
                if(j>=len){
                    break;
                }
                if(s[i]!=s[j]){
                    dp[i][j]=false;
                }else if(j-i<3){
                    dp[i][j]=true;
                }else dp[i][j]=dp[i+1][j-1];
                if(dp[i][j]) cont++;
            }
        }
        return cont;
    }
};
```

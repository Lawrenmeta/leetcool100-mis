用法

| **特性** | **unordered_set (集合)** | **unordered_map (映射)** |
| --- | --- | --- |
| **存储内容** | 仅存储 **Key**（键） | 存储 **Key-Value** 对（键值对） |
| **元素形式** | `s.insert(val)` set.count() erase() | `m.insert({key, val})` 或 `m[key] = val` |
| **访问方式** | 检查是否存在，遍历所有元素cou'n | 通过 Key 查找 Value，或遍历键值对 |
| **典型用途** | **去重**、判断某个元素是否出现过 | **计数**、建立索引（如：姓名 -> 电话） |

定义变量最好不要使用函数名 如max。

## 题目

给定一个字符串 `s` ，请你找出其中不含有重复字符的 **最长 子串** 的长度。

**示例 1:**

```
输入:s = "abcabcbb"
输出:3
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。注意 "bca" 和 "cab" 也是正确答案。

```

**示例 2:**

```
输入:s = "bbbbb"
输出:1
解释:因为无重复字符的最长子串是 "b"，所以其长度为 1。

```

**示例 3:**

```
输入:s = "pwwkew"
输出:3
解释:因为无重复字符的最长子串是 "wke"，所以其长度为 3。
     请注意，你的答案必须是 子串的长度，"pwke" 是一个子序列，不是子串。

```

**提示：**

- `0 <= s.length <= 5 * 104`
- `s` 由英文字母、数字、符号和空格组成

## 代码

```cpp
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        int a=0;
        int z=0;
        unordered_set<char> map;
        for(int i=0;i<s.size();i++){
            while(map.count(s[i])){
                map.erase(s[z]);
                z++;
            }
            map.insert(s[i]);
            a = max(a,i-z+1);
        }
        return a;
    }
};
```

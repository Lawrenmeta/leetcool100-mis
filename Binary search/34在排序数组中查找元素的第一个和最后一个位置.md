二分查找确实不是那么容易写的，划定好中间的mid后，需要严肃地考虑什么时候应该+1，什么时候应该-1，这个过程可以是较为灵活的，写二分查找时最好有一个记录值，记录找到了关键的数值，然后最后的形式就是向右缩，向左缩这样子。

```cpp
int binarySearch(vector<int>& nums, int target) {
    int l = 0, r = nums.size() - 1;
    int ans = -1; // 用来记录满足条件的“候选人”

    while (l <= r) { // 始终使用 <=
        int mid = l + (r - l) / 2;

        if (check(mid)) { // check() 是判断条件
            ans = mid;    // 1. 只要满足条件，先记录下来！
            // 2. 根据题目要求，决定往哪边继续收缩
            // 如果要找左边界，就往左缩：r = mid - 1;
            // 如果要找右边界，就往右缩：l = mid + 1;
        } else {
            // 3. 不满足条件，就去另一边找
            // 如果 mid 太小：l = mid + 1;
            // 如果 mid 太大：r = mid - 1;
        }
    }
    return ans; // 循环结束后，ans 里存的一定是最终结果
}
```

## 题目

给你一个按照非递减顺序排列的整数数组 `nums`，和一个目标值 `target`。请你找出给定目标值在数组中的开始位置和结束位置。

如果数组中不存在目标值 `target`，返回 `[-1, -1]`。

你必须设计并实现时间复杂度为 `O(log n)` 的算法解决此问题。

**示例 1：**

```
输入：nums = [5,7,7,8,8,10], target = 8
输出：[3,4]
```

**示例 2：**

```
输入：nums = [5,7,7,8,8,10], target = 6
输出：[-1,-1]
```

**示例 3：**

```
输入：nums = [], target = 0
输出：[-1,-1]
```

**提示：**

- `0 <= nums.length <= 105`
- `109 <= nums[i] <= 109`
- `nums` 是一个非递减数组
- `109 <= target`

## 代码

```cpp
class Solution {
public:
    vector<int> searchRange(vector<int>& nums, int target) {
        // 1. 找左边界
        int leftIdx = findBound(nums, target, true);
        
        // 如果左边界是 -1，说明根本没找到，直接返回
        if (leftIdx == -1) {
            return {-1, -1};
        }
        
        // 2. 找右边界
        int rightIdx = findBound(nums, target, false);
        
        return {leftIdx, rightIdx};
    }

private:
    int findBound(vector<int>& nums, int target, bool isFirst) {
        int l = 0, r = nums.size() - 1;
        int res = -1;
        
        while (l <= r) {
            int mid = l + (r - l) / 2;
            
            if (nums[mid] == target) {
                res = mid; // 记录位置
                if (isFirst) {
                    r = mid - 1; // 找左边界，往左缩
                } else {
                    l = mid + 1; // 找右边界，往右缩
                }
            } else if (nums[mid] < target) {
                l = mid + 1;
            } else {
                r = mid - 1;
            }
        }
        return res;
    }
};
```

    }

一种是复习unordered_map用法之后的遍历，另一种使用的是**Boyer-Moore 投票算法**，使用阴阳相消的思想，如果一个数目是候选者，随意让这个候选者与其他候选者发生消解的时候，余下的性质始终不会改变，以此为出发，可以设计一种随时倒戈的算法，使用数目之间进行比较互相消除，如果一个元素超过一半，那么在阴阳变换中一定会使其数目达到最大，最后剩下的依旧是它。

## 题目

给定一个大小为 `n` **的数组 `nums` ，返回其中的多数元素。多数元素是指在数组中出现次数 **大于** `⌊ n/2 ⌋` 的元素。

你可以假设数组是非空的，并且给定的数组总是存在多数元素。

**示例 1：**

```
输入：nums = [3,2,3]
输出：3
```

**示例 2：**

```
输入：nums = [2,2,1,1,1,2,2]
输出：2

```

**提示：**

- `n == nums.length`
- `1 <= n <= 5 * 104`
- `109 <= nums[i] <= 109`
- 输入保证数组中一定有一个多数元素。

## 代码

```cpp
#include <unordered_map>
class Solution {
public:
    int majorityElement(vector<int>& nums) {
        unordered_map <int,int> matrix;
        for(int i=0;i<nums.size();i++){
            matrix[nums[i]]++;
        }
        int max=0;
        int a=nums[0];
        for(auto t:matrix){
            if(t.second>max){
                 max=t.second;
                a=t.first;
            } 
        }
        return a;
    }

};
```

```cpp
class Solution {
public:
    int majorityElement(vector<int>& nums) {
        int consistant=nums[0];
        int times=1;
        for(int i=1;i<nums.size();i++){
            if(consistant==nums[i]) times++;
            else{
                if(times==0) consistant=nums[i];
                else times--;
            }
        }
        return consistant;
    }
};
```
